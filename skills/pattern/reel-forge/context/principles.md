## Principles
source: reel-forge/SKILL.md

**P1. One VOICE, held the whole SCRIPT.**
Reason: Voice is argument structure, not vocabulary. Two argument structures in one Reel produce cognitive dissonance that kills trust. Falsifiable: a script that switches from HORMOZI proof-first to NAVAL aphorism mid-way will feel incoherent — the viewer cannot extract one clear takeaway. Apply: at the BOUND VOICE line, lock it. Any rewrite request that blends voices gets two separate scripts instead.

**P2. One concrete specific in the HOOK's first line.**
Reason: The scroll window is under 3 seconds. A number, named tool, place, or admission creates a pattern-interrupt that fires before the viewer's thumb moves. Falsifiable: a HOOK that opens with "Have you ever wondered…" will be scrolled past; a HOOK that opens with "I lost $47,000 in 6 months" will not. Apply: if the input has no concrete specific, emit NEEDS_CONTEXT before writing the HOOK.

**P3. FAST-LANE by default; full pipeline only on launch signal.**
Reason: A system that demands mode + framework + HOOK picks before output becomes the trap — 22 modules, zero shipped reels (SKILL.md). The full pipeline is correct for launches, not for regular content. Falsifiable: a session that negotiates voice before generating a SCRIPT will produce fewer shipped reels than one that auto-matches and ships. Apply: auto-match the VOICE from topic signal; fire full pipeline only when input contains "launch", "series", or "repurpose".

**P4. Copy argument structure, not words.**
Reason: A creator who copied HORMOZI's exact words got parody; the one who copied his argument structure (proof before promise) got conversion (SKILL.md). The mechanism is credibility transfer — structure signals reasoning, vocabulary signals imitation. Falsifiable: a HORMOZI SCRIPT that uses his phrases without the proof-first structure will read as pastiche. Apply: write the creator's own proof in the HORMOZI structure, never transplant Hormozi's sentences.

**P5. The HOOK is a contract: the SCRIPT must pay it off.**
Reason: A CONTRARIAN HOOK that opens "[belief] is wrong" creates a curiosity gap that the audience holds open until the mechanism arrives. If the SCRIPT does not deliver the mechanism, the gap closes with disappointment — not conversion. Falsifiable: a HOOK claiming "discipline is a lie" followed by a generic list of habit tips will produce STOP before the payoff. Apply: write the payoff mechanism in the SCRIPT before finalizing the HOOK. Reverse-engineer the HOOK from the mechanism, not the other way.
