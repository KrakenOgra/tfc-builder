## Repurpose Map
source: reel-forge/SKILL.md

**Trigger:** REPURPOSE fires when the full pipeline runs (input contains "launch", "series", or "repurpose"). FAST-LANE suppresses REPURPOSE.

**SCRIPT → Carousel (Domino step):**
- Slide 1: The HOOK line verbatim (same text, different format — creator recognition).
- Slides 2-4: The 3 mechanism beats from the HOLD section ([3-30s]). One claim per slide. No paragraphs.
- Slide 5: The flip/payoff from [30-45s], condensed to one sentence.
- Slide 6: The close + CTA (if the VOICE allows a CTA).
- Visual rule: on-screen text from the Reel maps to carousel slide headlines. Body copy = one sentence per slide.

**SCRIPT → Thread:**
- Tweet 1: The HOOK line, slightly expanded (add the "why this matters" in one sentence).
- Tweets 2-5: One SCRIPT beat per tweet. Each tweet must stand alone as a complete thought.
- Final tweet: The CTA or the payoff aphorism if NAVAL.
- Rule: Thread opening must differ from the Reel HOOK by at least the first five words — platform audiences overlap; exact duplication reads as spam.

**VOICE constraints in REPURPOSE:**
- NAVAL: carousel slides are aphorisms only. Thread has no CTA tweet. The final tweet is the most dense aphorism from the SCRIPT.
- HORMOZI: carousel slide 6 = CTA with the single action. Thread final tweet = same CTA, same action.
- BUILDER/CONTRARIAN: carousel slide 1 can keep the failure/belief as the headline. Thread opens with a different angle on the same mechanism.
