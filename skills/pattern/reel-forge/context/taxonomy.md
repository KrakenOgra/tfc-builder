## Categories
source: reel-forge/SKILL.md

**1. BUILDER VOICE (personal mode)**
A SCRIPT that argues from personal failure to earned principle. The HOOK opens on a specific loss or number. Use when the creator has a firsthand outcome story with dates, tools, or metrics.

**2. CONTRARIAN VOICE (personal mode)**
Opens with "[Common belief] is wrong." BOUND to one counterintuitive claim per Reel. Use when the topic has a dominant false assumption the audience already holds.

**3. HORMOZI VOICE (external persona)**
Proof-before-promise structure: concrete result first, mechanism second, single-action CTA last. Use when the topic is an offer or method with a measurable outcome. Do not blend with NAVAL — splitting them doubled save rate per SKILL.md.

**4. NAVAL VOICE (external persona)**
Dense aphorism SCRIPT with zero CTA. Each line is a standalone truth. Use when the topic is a timeless principle, not a tactic. NAVAL carries no CTA by design.

**5. GODIN VOICE (external persona)**
Story-opens, empathy-closes. Slow talking-head delivery, no list structure. Use when the topic is about identity or worldview, not a step-by-step method.

**6. JOBS VOICE (external persona)**
Context → contrast → reveal cadence with theatrical pausing. Use when launching an idea or tool the audience has not seen. Hedging is forbidden.

**7. FAST-LANE RECIPE (default execution mode)**
Pre-wired VOICE + HOOK + SCRIPT that ships without a pipeline decision. auto-match selects from topic signal. Default when no VOICE is named.

**8. FULL PIPELINE (launch execution mode)**
VOICE → HOOK → SCRIPT → SHOTS & EDIT → on-screen text → REPURPOSE. Use for launches or when all formats are needed from one session.

## When To Use
source: reel-forge/spec.yaml

- **BUILDER / CONTRARIAN / GODIN**: input is first-person or contains a specific story with "I."
- **HORMOZI / JOBS**: input contains a number, result, or product name — proof-first structure selected.
- **NAVAL**: input is a principle with no tactical step and no CTA candidate — zero-CTA path enforced.
- **FAST-LANE**: topic arrives without a named VOICE — DONE in one pass, no mode negotiation.
- **FULL PIPELINE**: input contains "launch", "series", or "repurpose" — all five ## sections fire.
